SafeRadiusDC

1. Introduction

This API is designed to provide easy access to the data used for the map without requiring a download, which you can do here. Additionally, it offers filtered crime data.

2. Base URL

All API requests start with this base URL:

http://127.0.0.1:5000/api

3. Available Endpoints

Here are the main endpoints you can use:

  • GET /crimes: Retrieve crime data.

Using cURL

            -d "SHIFT=z"

Using JavaScript (Fetch API)

fetch('https://api.example.com/v1/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

5. Example Response

{
    "status": "success",
    "data": {
        "message": "Hello, World!"
    }
}

6. Error Handling

If something goes wrong, the API will return an error response:

{
    "status": "error",
    "message": "Endpoint not found"
}
